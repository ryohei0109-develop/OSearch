# ドキュメント検索CLIツール 要件定義・設計書

## 1. 概要

### 1.1 目的

.docx / .xlsx / .pptx ファイルの内容をJSON形式でインデックス化し、高速検索を実現するCLIツールを提供する。

### 1.2 背景

* 社内ドキュメントの検索性が低い
* ファイル名ではなく「中身」で検索したいニーズ
* ExcelやPowerPointの図形内テキストも検索対象にしたい

---

## 2. スコープ

### 2.1 対象ファイル

* Word（.docx）
* Excel（.xlsx）
* PowerPoint（.pptx）

### 2.2 対象データ

* 本文テキスト
* セル内容（Excel）
* オートシェイプ内テキスト（可能な範囲）

---

## 3. システム構成

```
[入力ディレクトリ]
        ↓
[インデクサ]
  - ファイル解析
  - JSON生成
        ↓
[index.json]
        ↓
[検索CLI]
  - 正規表現検索
  - 結果出力
```

---

## 4. CLI仕様

### 4.1 コマンド一覧

#### ■ インデックス作成

```
mycli index <target_directory>
```

##### 動作

* 指定ディレクトリ配下を再帰的に走査
* 対象ファイルを解析
* index.json を生成

---

#### ■ 検索

```
mycli search "<keyword>"
```

##### オプション

| オプション         | 内容       |
| ------------- | -------- |
| --regex       | 正規表現検索   |
| --ignore-case | 大文字小文字無視 |

##### 出力例

```
[1] C:\docs\sales.xlsx (Sheet: 売上管理, Cell: B12)
[2] C:\docs\plan.pptx (Slide: 3)
[3] C:\docs\contract.docx (Page: 2)
```

##### クリック動作

* ファイルパスをそのまま出力
* ターミナル上でクリック可能（環境依存）

---

## 5. JSONスキーマ

### 5.1 全体構造

```json
[
  {
    "file": "path/to/file.xlsx",
    "type": "excel",
    "contents": [...]
  }
]
```

---

### 5.2 contents構造

```json
{
  "location": {
    "sheet": "Sheet1",
    "cell": "A1",
    "slide": null,
    "page": null,
    "shape_id": "drawing1"
  },
  "content": "テキスト内容"
}
```

---

### 5.3 ファイル種別ごとの拡張

#### Excel

```json
{
  "sheet": "Sheet1",
  "cell": "A1"
}
```

#### PowerPoint

```json
{
  "slide": 3,
  "shape_id": 5
}
```

#### Word

```json
{
  "page": 2,
  "paragraph_index": 10
}
```

## 7. ファイル解析設計

### 7.1 PowerPoint

* python-pptxを使用
* shape.text から取得
* スライド番号を付与

---

### 7.2 Excel

#### セル

* openpyxlで取得

#### オートシェイプ

* xl/drawings/*.xml を解析
* a:tタグからテキスト抽出

---

### 7.3 Word

#### 本文

* python-docxで取得

#### 図形

* word/document.xml を解析
* w:tタグから抽出

---
